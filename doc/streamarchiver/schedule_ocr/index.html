Image → schedule OCR.

Several schedule sources publish the week as an image (a Twitch offline banner, a YouTube community post, a pinned tweet). We read those by shelling out to an LLM CLI — by default the claude CLI — with a strict OCR→JSON prompt, then map the returned events to ScheduleSegments. This ports the user’s working scripts/decode-schedule-img.ps1 into the app.

Everything here fails soft: a missing CLI, an auth prompt, a non-zero exit, or unparseable output all yield None so a transient OCR failure never wipes a previously-stored schedule.
