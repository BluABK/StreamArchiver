Build a Twitch channel’s third-party emote lookup: case-sensitive emote code → resolved on-disk image path. Reads the per-channel BTTV/FFZ/7TV manifests once (called on popup-open, not per message) and resolves each entry to its file in the per-channel or shared-global cache, keeping only those that exist.

Precedence on a code defined by multiple providers: 7TV > BTTV > FFZ — we insert in that order with or_insert, so the first (highest-priority) provider to define a code wins and later duplicates don’t clobber it.
