Derive the chat sidecar path from a recording’s output path. Locate a recording’s chat sidecar. yt-dlp’s live_chat writer appends to the -o value (keeping the video extension), so the YouTube sidecar is <output_path>.live_chat.json (e.g. clip.mkv.live_chat.json) — not a simple extension swap. The Twitch native logger instead replaces the extension (clip.chat.jsonl). We try both forms, plus the legacy pre-.cache YouTube name (clip.ts.live_chat.json).
