fn parse_youtube_schedule(body: &str) -> Vec<ScheduleSegment>

Extract upcoming scheduled streams from a YouTube /streams page. Each upcoming entry’s videoRenderer carries upcomingEventData.startTime (unix seconds) plus the title; we walk ytInitialData for those. Best-effort — returns an empty vec if the page shape changes.
