pub const TOGGLEABLE_JOBS: &[(&str, &str)];
Expand description

Periodic jobs that can be enabled/disabled from the Background view, as (display name, settings key). The display name matches the mark_job name so the UI can join a toggle with its next-run estimate. Default: enabled (get_setting(key) != "0"); each loop skips its work when disabled.
