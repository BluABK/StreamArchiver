fn prepare_and_load_icon(src: &Path) -> Option<HICON>
Expand description

Downscale src to 48×48 with Lanczos3, write an ICO to appdata, and load it via LoadImageW. Returns None on any failure (bad PNG, save error, …). Using LoadImageW with an ICO file avoids the manual BGRA flip and premultiply dance required by CreateBitmap/CreateIconIndirect.
