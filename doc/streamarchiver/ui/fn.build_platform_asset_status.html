Build the Properties window’s per-platform asset-status rows. Runs the blocking filesystem I/O once on open (the result is cached in channel_asset_status); see PlatformAssetStatus for why this must not run per frame. Logic mirrors what the status grid used to compute inline: icon/banner presence + archived-variant counts, a depth-2 badge-dir count, the Twitch emote-dir file count plus each third-party manifest length, and the last-fetched stamp.
