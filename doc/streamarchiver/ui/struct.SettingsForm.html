§twitch_client_id: String§twitch_client_secret: String§google_client_id: String

Google OAuth client (TV/device type) for “Connect YouTube” → subscriptions import. Separate from the YouTube Data API key (which can’t read user data).

§google_client_secret: String§youtube_api_key: String§youtube_api_detect: bool

Per-operation opt-ins to use the YouTube Data API key instead of scraping (each costs quota — see the Settings section).

§youtube_api_schedule: bool§youtube_api_quota_cutoff: String

Daily quota cutoff for the YouTube Data API (units). Empty = default (9000).

§youtube_search_quota_cutoff: String

Daily search.list query cutoff (queries). Empty = default (90).

§kick_client_id: String§kick_client_secret: String§default_output_dir: String§max_concurrent_downloads: String§download_auth_method: String

Global download-auth default: “none” or “cookies”.

§cookies_browser: String

Browser to read cookies from (yt-dlp --cookies-from-browser).

§cookies_profile: String

Optional browser profile/session (the part after browser:).

§websub_vps_url: String

YouTube WebSub VPS relay (yt-websub) — base URL, bearer token, poll secs.

§websub_token: String§websub_poll_secs: String§filename_media_info: MediaInfoMode

When to probe captures for the {resolution}/{fps}/… filename variables.

§date_fmt: DateFmt

How dates/timestamps are displayed throughout the UI.

§short_ts_fmt: String

chrono format string used for the compact timestamp mode (K_SHORT_TS_FMT).

§ytdlp_default_args: String

Global extra arguments prepended to every yt-dlp invocation (all monitors). Per-monitor extra_args are appended after these, so they take precedence.

§ytdlp_binary_path: String

Explicit path to the system yt-dlp binary; empty ⇒ yt-dlp on PATH.

§sabr_binary_path: String

Path to the SABR dev-build yt-dlp (for YouTube capture-from-start). Empty ⇒ SABR disabled (capture-from-start uses the system binary’s normal path).

§sabr_enabled: bool

Master toggle: use the SABR build for YouTube capture-from-start.

§sabr_format: String

SABR format selector + extractor-args preset.

§sabr_extractor_args: String§sabr_deep_rewind: bool

Experimental deep-rewind toggle (appends enable_live_deep_rewind=true).

§sabr_raw_args: String

Manual raw SABR args; non-empty overrides the format+extractor-args preset.

§sabr_pot_args: String

PO-token-provider --extractor-args (e.g. bgutil) for the SABR command.

§dash_format: String

DASH-companion format selector used by dual (SABR+DASH) capture.

§discord_token: String

Discord user token + whether to import stream schedules from Discord events (opt-in; automating a user token is against Discord’s ToS).

§discord_schedule: bool§ocr_command: String

Image→schedule OCR pipeline (shells out to an LLM CLI). ocr_command is the executable; ocr_model/ocr_fallback_model the primary + retry models; ocr_timezone/ocr_offset the timezone/UTC offset to assume for banner times. Empty fields fall back to the built-in defaults.

§ocr_model: String§ocr_fallback_model: String§ocr_timezone: String§ocr_offset: String§ocr_max_budget: String

Per-call USD budget cap passed as --max-budget-usd (empty = no cap).

§ocr_timeout_secs: String

Process timeout in seconds (empty/0 = default 150 s).

§ocr_effort: String

Effort level passed as --effort (empty = omit; low/medium/high/xhigh/max).

§dialog_icon: String

File path to a PNG used as the main icon in crash and freeze dialogs. Empty = standard Windows error/warning icon. Requires a restart to take effect.

§schedule_title_fill: bool

Global “go to the next schedule source when an event has no title” toggle: after a winner is found, keep querying lower-priority sources to fill in blank titles (e.g. a Twitch schedule with times but no titles).

§youtube_community_max_posts: String

How many recent YouTube community posts to scan for a schedule image (backlog depth). Empty = built-in default (5). Per-channel override in channel Properties.

§remux_embed_thumbnail: bool

Embed the thumbnail sidecar as MKV cover art on remux.

§remux_embed_title: bool

Embed a title metadata tag in the MKV on remux.

§remux_title_template: String

Template used to generate the MKV title tag.

§remux_embed_subs: bool

Embed subtitle sidecar files as MKV subtitle streams on remux.

§file_split_enabled: bool

Split output files into per-type subdirectories.

§file_split_videos: String§file_split_subs: String§file_split_chat: String§file_split_thumbs: String§file_split_logs: String§fetch_thumb_embed: bool

Checkbox for “fetch missing thumbnails” in the Maintenance section.

§maintenance_filename_preset: String

Selected preset template for the “Set filename default” row in Maintenance.

§maintenance_apply_all: bool

Apply the preset to all existing monitors when “Set as Default” is clicked.

§media_player_path: String

Path to the media player binary (e.g. C:\Progs\mpv\mpv.exe).
