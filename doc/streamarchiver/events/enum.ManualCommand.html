Expand description

On-demand recording commands from the UI to the download supervisor.

Variants§

§

Start

Check the channel now and, if live, start recording (bypassing backoff). notify_offline: toast an error when the channel turns out not to be live. Set to true for explicit UI button clicks; false for automatic triggers (WebSub events) so a “not live” outcome is silently discarded.

Fields

§id: i64
§notify_offline: bool
§

Stop(i64)

Abort the channel’s active recording.

§

StartVideo(i64)

Begin downloading a queued on-demand video (by video id).

§

StopVideo(i64)

Abort an in-flight on-demand video download (by video id).

§

StopChat(i64)

Abort the live-chat sidecar download for a monitor (by monitor id). The chat download runs independently of the video recording so it needs its own stop command.

§

RefetchAssets(i64)

Force a channel-asset refetch for a monitor (by monitor id), ignoring the 24h freshness stamp and the per-monitor fetch_chat_assets toggle.

§

ReRemux

Re-remux a captured .ts file to MKV in the background (for recordings where the automatic remux failed at finalization time). On success the source TS is deleted and recording.output_path is updated to the MKV.

Fields

§rec_id: i64
§capture: PathBuf
§final_: PathBuf
§

ReRemuxAll

Re-remux all recordings that still have a .ts source file.

§

EmbedMissingThumbnails

Embed the thumbnail sidecar into all MKV files that don’t already have one.

§

FetchMissingThumbnails

Download missing thumbnails for recordings; if embed is true, immediately embed them into the MKV after download.

Fields

§embed: bool
§

ReorganizeAll

Reorganize all recordings according to the current crate::models::SubdirConfig.

§

ReorganizeTake(i64)

Reorganize all recordings belonging to a specific take group (by recording id).

§

ReorganizeMonitor(i64)

Reorganize all recordings belonging to a monitor.

§

ReorganizeChannel(i64)

Reorganize all recordings belonging to a channel.

§

RenameRecording

Rename a recording’s output file using the given new filename stem.
