Filesystem locations for config, database, and default recording output.

Functions§

asset_cache_dir
Root of the centralised asset cache (alongside the DB, under the data dir). e.g. %APPDATA%\StreamArchiver\data\asset-cache\ on Windows.
data_dir
Resolve (and create) the application data directory, e.g. %APPDATA%\StreamArchiver\data on Windows.
db_path
Path to the SQLite database. Kept on local disk (WAL requires it).
default_output_dir
Default per-channel output directory: the user’s Videos folder if available, otherwise a recordings folder under the app data dir.
logs_dir
Directory for rotating log files, e.g. %APPDATA%\StreamArchiver\data\logs\.
platform_assets_dir
Platform-wide shared asset cache for deduplicated emote images and global badges. e.g. %APPDATA%\StreamArchiver\data\asset-cache\platform_assets\.
