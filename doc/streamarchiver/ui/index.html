Module ui 

Source
Expand description

The on-demand egui window: channel table, add/edit form, and settings.

Runs reactive (repaints only on input/events). The tray thread wakes it via Context::request_repaint. Closing the window hides it to the tray; the tray “Quit” item triggers a real close.
