pub fn reattach_fixups(spawn_build: &str, row: &mut DetachedRow) -> bool
Expand description

Apply any fixups needed to safely re-attach to row, which was spawned by build spawn_build (already known to differ from the running build). Mutates row in place (e.g. rewriting a path an old build used) and returns whether any fixup was applied. Each fixup should info!-log what it did.

No fixups exist yet — the current build is the only one that has ever written the registry. The version parse below is the seam: add migrations keyed by the spawning build, e.g. if older_than(version, "0.2.0") { rewrite_capture_path(row); return true; }.
