streamarchiver

Module downloader 

Source
Expand description

Download supervisor + per-tool adapters.

When the scheduler reports a monitor live, the supervisor (bounded by a global concurrency semaphore) spawns the configured tool as a child process, captures its stderr into a ring buffer, waits for exit, classifies the outcome, optionally remuxes TS -> MKV, and records the run in the store. A Win32 Job Object guarantees the whole process tree is killed on stop/exit.

Default container is MKV (never MP4): streamlink records to .ts then remuxes losslessly to .mkv; yt-dlp merges straight to .mkv.
